# Retro Video Game Gallery

A family‑focused static single‑page application that showcases classic video games.

- Responsive gallery grid of game cards
- Search and filter by platform and genre
- Favorites (persisted via localStorage)
- Data stored in a static `games.json` with cover images in `images/`
- Deployed on GitHub Pages

## Development

1. Clone the repo
2. Install dependencies (if any)
3. Run the development server (e.g., `npm run dev` or similar)
4. Build for production and push to `gh-pages`.
